# Apex Gym & Fitness Center Website

A simple single-page HTML website built for a fitness center called Apex Fitness. This project demonstrates responsive page structure, navigation, styled sections, and a small BMI calculator using vanilla HTML, CSS, and JavaScript.

## Project Contents

- `website.html` - The full landing page with header navigation, hero section, about section, programs, trainers, BMI calculator, membership plans, and footer.

## Features

- Sticky top navigation bar
- Responsive layout using CSS grid and media queries
- Branded hero section with fitness messaging
- Program and trainer overview cards
- BMI calculator with live input validation and category output
- Clean, modern styling with accent colors and hover effects

## How to View

1. Open `website.html` in your web browser.
2. Scroll through the sections or use the navigation links to jump to each section.
3. In the BMI section, enter weight in kilograms and height in centimeters, then click `Calculate Now`.

## Notes

- The page is fully self-contained and does not require a web server.
- All styling and scripting are embedded directly in `website.html`.
- This is ideal as a portfolio project or a small static landing page for a gym or fitness center.

## Suggested Improvements

- Separate CSS into an external stylesheet
- Add a contact form for membership inquiries
- Enhance accessibility with explicit labels and ARIA attributes
- Add animations or a slideshow for featured programs
